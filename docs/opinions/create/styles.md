# Opinion: Creative Styles

> Visual and aesthetic style guidelines

## Design Constraints

- Maintain consistency within a project or brand
- Establish visual hierarchy through contrast, scale, and spacing
- Choose typography that supports readability and tone
- Use color intentionally to convey meaning and mood

## Requirements

- Define a style guide or reference before execution
- Document color palette, typography, and spacing rules
- Ensure assets are reusable and organized
- Consider context and medium (print, screen, mobile)

## Anti-Patterns

- Mixing too many fonts or colors without purpose
- Inconsistent spacing and alignment
- Ignoring accessibility (contrast, legibility)
- Style choices that fight the content

## Assumed Context

- Work intended for an audience
- Visual coherence matters
- May need to scale or adapt across formats

## Summary

Intentional, consistent visual choices that serve the content and audience.
