# Praxis Governance

This directory contains [Praxis](https://github.com/jayers99/praxis-ai) governance files that track the lifecycle stage and decisions for this project.

## Files

| File | Purpose |
|------|---------|
| `praxis.yaml` | Current state (domain, stage, privacy, environment) |
| `CAPTURE.md` | Raw inputs and prior art gathered |
| `SENSE.md` | Patterns and themes organized |
| `EXPLORE.md` | Options evaluated and spikes run |
| `SHAPE.md` | Final form defined |

## Current Stage

See `praxis.yaml` for the current lifecycle stage.
